# 外围赌注 / 比赛操控 作为因子 —— 可行性评估

整理日期：2026-06-13 ｜ 维护者：Claude

老板诉求:把外围赌注当因子,识别"可能被操控/打假球"的比赛。下面是诚实评估,不夸大。

## 一、结论先说

1. **能做的、且已做**:用"模型 vs 市场盘口分歧"做一个**市场信号因子**(`src/marketSignal.js`,已上线)。盘口和模型分歧大时提示"市场可能已计入伤停/阵容/资金面信息,谨慎看待本场"。这是真实有用的。
2. **不能做、也不该假装能做**:用单一快照赔率"检测假球"。真正的操控检测是 FIFA / Sportradar UFDS 这类机构的活,需要**多家博彩、时间序列**的赔率异动数据 + 反洗钱级别的资金流,我们没有。
3. **红线**:**绝不**给任何真实球队/比赛打"假球/被操控"标签——这是诽谤,且误报率极高。我们能说的最强表述就是"市场信号异常,值得留意"。代码里已硬性保证 note 不含"假球/操控/fix"字样,并有测试守着。

## 二、为什么世界杯几乎不用担心操控

- **最高曝光度**:世界杯是全球监控最密的赛事,FIFA 有专门 integrity 监测 + Sportradar 实时盯盘,球员身价/曝光极高,操控的风险回报比极差。
- 学术与行业共识:假球绝大多数发生在**低级别、低曝光联赛**(身价低、监控弱),不是世界杯。
- 所以对一个 2026 世界杯工具,"操控检测"的实战价值很低;**真正有价值的是同一信号的另一面:识别"信息差资金"**(伤停/首发泄露/内幕),这正是市场信号因子在做的。

## 三、我们能算的 vs 真操控检测需要的

| 维度 | 我们现在有 | 真操控检测需要 |
|---|---|---|
| 赔率快照 | ✅ DraftKings 单点(经 ESPN) | 多家博彩 |
| 时间序列异动 | ❌ | ✅ 赛前数小时/数天的赔率漂移曲线 |
| 市场覆盖 | 1 家 | 几十家 + 亚盘/大小球 |
| 资金流 | ❌ | 投注量/异常大额 |
| 判定 | 模型-市场分歧度 | 多源异动 + 人工 integrity 复核 |

**市场信号因子**用我们手上的数据做到了"分歧度"这一层——足够提示用户谨慎,够不到"判定操控"。

## 四、已落地(`src/marketSignal.js`)

- `marketSignal(modelProb, marketProb)` → 返回分歧度 + 等级(aligned <0.08 / notable 0.08–0.15 / strong >0.15)+ 市场更偏向哪一侧 + 提示文案。
- 每场有市场数据的比赛行带 `marketSignal` 字段;强分歧 = 给用户的"谨慎"软标记,**不是操控指控**。
- 当前只有 B-1/D-1 有市场快照;接入更多赔率后覆盖更全。

## 五、若要往前走(需要老板决策)

- **加时间序列**:接一个赔率聚合源(The Odds API 免费档 500 次/月起;或 OddsPortal 抓取),存赛前赔率漂移,才能做"异动"而非"分歧"。这是把"市场信号"升级成"赔率异动雷达"的前提。
- **定位仍是娱乐/研究**:即便加了异动,对外表述也只能是"市场异动,供参考",绝不升级成"操控判定"。这条对老板一人公司的法律风险是硬约束。

## 信源 / 方法

- 行业做法:Sportradar UFDS、FIFA Integrity(多源时间序列赔率异动);本评估据其公开方法论概述。
- 本仓库:marketFusion.js(去水隐含概率)、marketSignal.js(分歧度)。
