# 预测准确性研究：如何更准 + 场外因子清单

整理日期：2026-06-12 ｜ 维护者：Claude（共享给 Codex 和未来 agent）

目标：回答两个问题——(1) 什么方法能让胜平负预测更准；(2) 场外因子有哪些、各自影响多大、哪些值得进模型。

---

## 一、什么方法预测最准（按证据强度排）

1. **市场赔率是最强单一预测源。** 大量研究反复验证：去掉抽水后的博彩公司隐含概率,长期 Brier 分数优于几乎所有公开统计模型(包括 Elo 和泊松)。原因是赔率聚合了伤病、训练情报、内部消息等模型拿不到的信息。
   - 落地:我们已经能从 ESPN 拿 DraftKings 实时赔率。**模型概率与市场隐含概率加权融合**(如 50/50 起步,再按各自历史 Brier 调权)是性价比最高的单项升级。
2. **泊松类进球模型(Maher 1982 / Dixon-Coles 1997)是统计模型的主流骨架。** 从两队攻防强度推期望进球 λ,自然产出完整比分分布——胜平负、净胜球、爆冷概率全部来自同一个模型,不需要单独的 drawBias 启发式。Dixon-Coles 修正了低比分(0-0/1-1)的相关性,这正是世界杯小组赛常见区间。
3. **Elo 本身已经不差,但有已知盲区**:
   - 对国家队,eloratings.net 这类带进球差权重的 Elo 长期表现接近 FIFA 排名之上;
   - 盲区 = 阵容突变(主力伤停)、风格相克、比赛动机(末轮轮换)。
4. **xG(期望进球)比比分本身更有预测力。** 复盘时用 xG 而不是比分评估"模型偏差"——韩国 1.84 vs 捷克 0.81 的 xG 说明 2-1 不是侥幸。比分含大量噪声,xG 是更干净的信号。
5. **集成(ensemble)稳定优于单模型。** 我们现有的四层结构(Elo→状态→政策→市场)方向正确;关键是给每层用真实数据喂参,并用累计 Brier 分数验证每层是否真的在加分。

## 二、场外因子清单(2026 本届适用)

按影响量级和可建模性排序:

### A 级:影响大、可量化、应该进模型

| 因子 | 量级证据 | 2026 具体情况 |
|---|---|---|
| **东道主/主场效应** | 世界杯历史东道主全部小组出线;主场效应约折合 +80~100 Elo | 三东道主不等价:墨西哥(Azteca 9万人+海拔)≫ 美国(场馆多、中立感强)> 加拿大 |
| **海拔** | 高海拔主队对平原客队有显著额外优势(BMJ 高海拔研究);疲劳加速、冲刺能力下降 | Mexico City 2,200m、Guadalajara 1,566m、Monterrey ~540m;美加场馆全部 <320m。**凡在墨西哥城/瓜达拉哈拉踢的比赛,适应过的球队(墨西哥、南美高原队)应加成** |
| **红牌/少打一人** | 文献共识:少 1 人每 90 分钟约损失 1.4 球净值 | 揭幕战 1 场 3 红(开幕战历史纪录),说明本届判罚尺度偏严——红牌频率可能高于历届,爆冷通道变宽 |
| **休息天数差** | 少休 1-2 天在淘汰赛阶段有可测影响 | 48 队新赛制赛程更长,末段休息差会拉大 |
| **小组末轮动机不对称** | 已出线队大轮换,垫底队放飞——末轮预测误差历史上最大 | 复盘时给末轮单独标注,不要把末轮失准归咎模型本身 |
| **旅行距离/时区** | 2026 是史上跨度最大的一届(三国 16 城) | 各队小组赛被分配到区域簇,但淘汰赛可能横跨时区——淘汰赛阶段把累计飞行距离做成修正项 |

### B 级:影响真实但难量化,定性跟踪

- **气候/热**:6-7 月北美午后场(为欧洲电视黄金档)体感 35°C+;耐热球队(中东、中美洲)受益。
- **裁判倾向**:主场观众影响补时和判罚的研究很扎实(Garicano et al.);世界杯裁判中立性较强,影响小于联赛但非零。
- **阵容缺席**:停赛累积(2 黄停赛)+伤病。这是市场赔率领先模型的主要信息差,我们靠每日拉赔率间接吸收。
- **教练临场/换人质量**:韩国第二球来自替补 Oh Hyeon-gu——换人贡献是 Elo 完全看不到的层。
- **门将状态**:单场被射正转化率波动极大,小样本里门将就是方差本身。

### C 级:情绪面,只做解释不做预测

- 国民情绪/政治事件(Edmans 2007:输球→次日该国股市 -49bp,情绪是真实的,但反向预测比赛没用);
- 球星个人叙事(Jiménez 35 岁首个世界杯进球这类故事影响舆论热度,不影响下一场概率)。

## 三、对本项目的落地优先级

1. **[P0] 复盘闭环**:每场完赛记录 模型概率 vs 市场概率 vs 实际结果 + xG,累计 Brier 分数(模型 vs 市场双轨),72 场小组赛打完就有结论"我们离市场差多少"。→ 见 `docs/match-reviews/`。
2. **[P0] 市场融合**:ESPN/DraftKings 赔率转隐含概率,与模型概率加权平均后作为"展示概率"。
3. **[P1] 海拔修正**:墨西哥城/瓜达拉哈拉场次,给适应队 +,给平原客队 -(从场馆数据驱动,不拍脑袋)。
4. **[P1] 动态 Elo**:完赛后按 K=60 更新,模拟剩余赛程用更新后的值。
5. **[P2] 泊松化**:用 Elo 差映射 λ,替换现有比分启发式,统一比分/平局/净胜球。
6. **[P2] 红牌情景**:本届判罚严,模拟里加入每场 ~30% 出红牌的情景分支(参数用本届实际频率滚动更新)。
