## A-1 墨西哥 2-0 南非(Estadio Banorte/Azteca,墨西哥城)

**赛前模型**:墨西哥 77.7% / 平 15.4% / 南非 6.9% → modal=墨西哥胜 ✅ 命中

**进程**:9' Quiñones 抓 Sithole 后场失误低射穿裆 1-0;下半场 Jiménez 推射 2-0(35 岁生涯首个世界杯进球)。全场 3 张红牌(南非 2、墨西哥 1),开幕战历史纪录;49' Sithole 红牌后比赛失去悬念。

**数据**:控球 60.5/39.5;射门 16/3;射正 4/2。

**模型 vs 实际的出入**:
1. **方向完全正确,但支配程度被低估**。370 Elo 差给出 77.7%,实际南非全场仅 3 次射门——真实压制接近"90%+ 一边倒"。高海拔(2,200m)+9 万主场是 Elo 之外的额外杠杆,印证研究文档里"墨西哥城场次需要海拔修正"的判断。
2. **红牌完全在模型外**。3 红是本场最大异常值;若南非 11 人踢完,比分大概率仍输但射门数不会这么难看。
3. **战术注脚**:南非中场出球失误直接送了第一球——低 Elo 队在高压下的失误率,泊松模型可以吃进去,纯 Elo 吃不进去。

## A-2 韩国 2-1 捷克(Estadio Akron,瓜达拉哈拉)

**赛前模型**:韩国 45.3% / 平 25.2% / 捷克 29.6% → modal=韩国胜 ✅ 命中(但只押了 45%)

**进程**:捷克 Krejčí 接 Coufal 大力手抛球头球先下一城;8 分钟后 Hwang In-beom 接 Lee Kang-in 直塞内切兜射扳平(67');79' 替补 Oh Hyeon-gu 绝杀 2-1。

**数据**:控球 61.7/38.5;射门 15/7;xG **1.84 vs 0.81**;Lee Kang-in 传球 37/37 全成功、对抗 10/14、创造 3 次机会。

**模型 vs 实际的出入**(这场信息量比 A-1 大):
1. **模型明显低估了韩国**。45.3% 是"略优";xG 1.84-0.81 和控球 62% 说明真实优势接近 65-70% 量级。74 分 Elo 差没体现出两队风格代差:韩国传控+高位逼抢打直来直去的捷克,是结构性克制。**风格相克是纯 Elo 的固有盲区**(研究文档 B 级因子)。
2. **捷克唯一的威胁通道是定位球/手抛球**,进球正来自长手抛——“低控球队靠定位球得分”是经典模式,模型没有定位球维度,这类队的平/爆冷概率被系统性低估。
3. **球员层面**:Son 多次错失良机(锋线把握度方差)、Lee Kang-in 是全场最优(模型的球星池有他,方向对);**绝杀来自替补 Oh Hyeon-gu——换人/板凳深度完全在模型之外**,而 48 队长赛制下板凳价值会被持续放大。
4. **教训**:比分 2-1 看着惊险,xG 看其实是控制局。复盘一律以 xG 为主、比分为辅评估模型偏差。

## 复盘 SOP(每场执行)

1. 完赛后从 ESPN summary API 取:比分、进球、红牌、控球、射门、xG;
